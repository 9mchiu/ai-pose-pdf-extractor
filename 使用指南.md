# 使用指南

這份指南教你怎麼從零開始，把電腦準備好、跑起 **PDF → Word 網頁轉換器**，以及使用「數據指標萃取」工具。跟著步驟做即可，不需要寫程式。

---

## 〇、事前準備（只要做一次）

### 1. 安裝 Python
- Mac 通常已內建。打開「終端機」(Terminal)，輸入下面指令確認：
  ```bash
  python3 --version
  ```
  有出現版本號（例如 `Python 3.11.x`）就 OK。若沒有，到 https://www.python.org 下載安裝。

### 2. 準備 Anthropic API Key
- 到 https://console.anthropic.com 登入 → 建立一組 API Key（`sk-ant-...` 開頭）。
- 這把鑰匙等於你的用量帳單，**不要貼到網路上或截圖分享**。

---

## 一、PDF → Word 網頁轉換器

把 PDF 拖進網頁，自動轉成 Word：**文字變文字、圖片保留為圖片**。掃描版 PDF 也能讀。

### 步驟 1：打開終端機，進到工具資料夾
```bash
cd ~/ai-pose-pdf-extractor/pdf_to_word_web
```

### 步驟 2：建立環境並安裝套件（只要做一次）
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
> 之後每次重開終端機要再跑一次 `source .venv/bin/activate` 來啟用環境。

### 步驟 3：填入你的 API Key
```bash
export ANTHROPIC_API_KEY="把你的_sk-ant-金鑰_貼在這裡"
```

### 步驟 4：啟動網頁
```bash
python app.py
```
看到 `http://127.0.0.1:5000` 就代表啟動成功。

### 步驟 5：開始轉換
1. 打開瀏覽器，網址列輸入：**http://127.0.0.1:5000**
2. 把 PDF **拖進中間的框**（或點框選檔案）。
3. 等它處理（頁數多時約 1~2 分鐘），完成後會**自動下載** Word 檔。
4. 下載後畫面會**自動清空**，可以接著拖下一份。

### 要關掉時
回到終端機，按 `Ctrl + C` 即可停止。

### 隱私說明
- 檔案只在你自己的電腦上處理，**不會存到硬碟、也不會上傳到任何網站**。
- 網頁只在本機執行（`127.0.0.1`），API Key 不會外洩到前端。

## 二、數據指標萃取工具（選用）

把一整批 PDF 報告裡的數據（MPJPE、FPS、精度…）抽成一張 CSV 表，方便跨報告比較。

### 步驟
```bash
cd ~/ai-pose-pdf-extractor
pip install anthropic
export ANTHROPIC_API_KEY="你的金鑰"
```
1. 把要分析的 PDF 全部放進 `~/ai-pose-pdf-extractor/pdfs/` 資料夾。
2. 執行：
   ```bash
   python3 extract_pose_metrics.py
   ```
3. 完成後看 `output/` 資料夾：
   - `metrics.csv`：可用 Excel 打開的比較表
   - `results.json`：完整結果

## 三、常見問題

| 狀況 | 解法 |
|------|------|
| `command not found: python3` | 尚未安裝 Python，到 python.org 安裝 |
| 網頁顯示「伺服器未設定 ANTHROPIC_API_KEY」 | 忘了做步驟 3 的 `export`，或重開終端機後要再設一次 |
| `ModuleNotFoundError` | 忘了啟用環境，先跑 `source .venv/bin/activate` |
| 轉換很慢 | 掃描頁是圖片、頁數多時較慢也較花費，屬正常 |
| 圖片位置有點跑掉 | 目前為 v1，可在 `app.py` 調 `RENDER_DPI`；或回報給我調整 |
| 想換個位置放資料夾 | 整個 `ai-pose-pdf-extractor` 資料夾可任意搬移 |

## 四、調整設定（進階，選用）

打開 `pdf_to_word_web/app.py`，最上方可調：
- `RENDER_DPI`（預設 200）：頁面渲染清晰度，調高更清楚但更慢/更貴。
- `MAX_IMG_WIDTH_IN`（預設 6.0）：Word 內嵌圖片最大寬度（英吋）。
- `MODEL`：使用的模型。
